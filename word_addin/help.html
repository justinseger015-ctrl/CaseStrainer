BriefCheck Help

About BriefCheck

BriefCheck is a tool that helps legal professionals identify potentially hallucinated case citations in legal briefs. It uses a novel approach based on the consistency of multiple AI-generated summaries of each cited case to determine whether a citation is likely to be real or hallucinated.

How It Works

BriefCheck works on the principle that when an AI model is asked to summarize a real case multiple times, the summaries will be highly similar to each other. In contrast, when asked to summarize a non-existent (hallucinated) case, the AI will generate significantly different summaries each time, resulting in low similarity scores.

The Process
  1. Citation Extraction: BriefCheck extracts case citations from your document using specialized legal citation parsers.
  2. API Verification: Each citation is checked against legal databases like CourtListener's Citation Lookup API.
  3. Summary Generation: For each citation, BriefCheck generates multiple summaries using an AI model.
  4. Similarity Calculation: The tool calculates the similarity between these summaries.
  5. Hallucination Detection: Citations with low similarity scores are flagged as potentially hallucinated.

Using BriefCheck

Basic Usage
  1. Open your legal brief in Microsoft Word.
  2. Click on the BriefCheck button in the Home tab.
  3. In the taskpane that appears, click "Analyze Document".
  4. Review the results, which will show any potentially hallucinated citations.
Advanced Settings
  • Number of summary iterations: More iterations increase accuracy but take longer. Default is 3.
  • Similarity threshold: Adjust this to control sensitivity:
    • Lower threshold (e.g., 0.5): More lenient, fewer false positives but might miss some hallucinations.
    • Higher threshold (e.g., 0.9): More strict, catches more hallucinations but might have more false positives.

Interpreting Results

Result Types
  • Likely Real: Citations with high similarity scores between summaries.
  • Potentially Hallucinated: Citations with low similarity scores between summaries.

For each citation, BriefCheck provides:

  • Detection method: Whether the citation was checked via API or summary comparison.
  • Confidence: How confident the system is in its determination.
  • Similarity score: The average similarity between generated summaries.
  • Generated summaries: The actual summaries used for comparison.

Limitations

BriefCheck is a tool to assist legal professionals, not replace their judgment. Please note:

  • The tool may occasionally produce false positives or false negatives.
  • Always verify flagged citations through traditional legal research methods.
  • The tool works best with standard citation formats and may miss non-standard citations.
